📘 PokéFetch

A responsive web app to explore Pokémon using the PokéAPI.
Built with HTML, CSS, and JavaScript, featuring search, sorting, pagination, and dark mode.

🚀 Features
🔍 Search Pokémon by name
🔤 Sort Pokémon (A–Z, Z–A)
📄 Pagination (40 Pokémon per page)
🌙 Dark mode toggle
⚡ Fast data fetching with async/await
📱 Fully responsive design
🛠️ Tech Stack
HTML5
CSS3 (Flexbox, Grid, Variables)
JavaScript (Vanilla JS)
PokéAPI
⚙️ How It Works
Fetches Pokémon data from PokéAPI
Displays them in a card layout
Supports live search and sorting
Uses pagination for better performance
Dark mode is handled via CSS variables
